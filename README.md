### Project prerequisites
1. Docker Compose ^2.15.1
2. Node 18.14.2 / npm 9.5.0
### Project Setup
1. Create `.env` file from `.env.sample` after providing the DB credentials.
2. `$ docker-compose up`
3. `$ npm ci`
4. `$ npm start`
